# Projeto 1 - CRUD de Produtos + Login

## 1. Sobre o Projeto
Este projeto é uma aplicação web em Java usando Spring Boot e Thymeleaf.  
Permite criar, listar, editar e deletar produtos, com login e área restrita para usuários.  
Foi desenvolvido como requisito da disciplina de Sistemas de Informação.  

**Funcionalidades principais:**
- CRUD completo da entidade Produto (nome, preço)
- Login com perfis ADMIN e USER
- Área restrita para usuários logados
- API REST para integração externa
- Frontend usando Thymeleaf, HTML, CSS e JavaScript
- Persistência de dados com MySQL

## 2. Tecnologias
- Java 17+, Spring Boot, Spring Web, Spring Data JPA, Spring Security
- Thymeleaf, HTML, CSS, JavaScript
- MySQL 8+, Maven
